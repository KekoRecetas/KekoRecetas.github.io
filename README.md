# KekoRecetas.github.io
Archivo personal de recetas y postres probados, con notas, fuentes y fotos.

## Qué es este proyecto
Este repositorio es un archivo culinario personal donde guardo recetas, principalmente de repostería, que he probado, junto con:
- Ingredientes
- Pasos de elaboración
- Fotos de los pasos y del resultado
- Notas personales sobre la receta
- Fuente que he seguido para la receta

## Objetivo
Tener una colección organizada de recetas que he ido probando, con mis propias anotaciones y mejoras.

## Organización
Cada receta se encuentra en su propia página o archivo dentro del repositorio.
